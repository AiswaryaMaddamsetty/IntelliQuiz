Quizzy is a modern, AI-assisted desktop application designed to generate, store, and manage quizzes easily. It uses Gemini AI to create intelligent, topic-based questions and stores quiz data locally using SQLite. The app is built with a user-friendly interface to help educators, students, and learners quickly generate quizzes without manual effort.

**Features**

AI-Generated Questions using Gemini

Topic-Based Quiz Creation

Multiple Question Types (MCQs, short answers, fill-in-the-blanks)

Local Quiz Storage with SQLite

User-Friendly Desktop UI

Automatic Saving of Quizzes for later use

Error Handling & Secure API Use

Lightweight & Fast Execution

**Tech Stack**

Python

Tkinter (GUI)

SQLite (Database)

Gemini API

dotenv (for secure API key configuration)

**How to Run the Application**
1. Clone this repository
git clone https://github.com/your-username/quizzy.git
cd quizzy

2. Install required dependencies
pip install -r requirements.txt

3. Add your Gemini API key

Create a .env file in the project root:

GEMINI_API_KEY=your_api_key_here

4. Run the application
python main.py


The Quizzy app will launch with a graphical interface where you can generate quizzes by entering a topic, difficulty level, or custom instructions.
